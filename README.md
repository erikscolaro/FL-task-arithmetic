# FL-task-arithmetic

## General sturcture of the repo

- checkpoints: contains the model checkpoints.
- data: contains the dataset and its splits.
- notebooks: contains the .ipynb to perform experiments.
- resutls: contains the experiments results.
- scripts: contains utility scripts for colab execution, as downloading the dataset, split the dataset etc. 
- src: contains models definition, hyperparameter configuration, custom optimizers etc.

> **Please note:** the dataset should't be stored on github! for that reason, the data folder content will be exluded by the .gitignore.
